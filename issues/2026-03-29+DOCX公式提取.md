# DOCX 数学公式提取（OMML → LaTeX）

## 背景
mammoth 转 Markdown 时完全丢弃 OMML 公式节点。论文中的数学公式在 LLM 审阅时不可见。

## 方案
复用已有 OpenXML 基建（documentXml + @xmldom/xmldom），新增 omml2mathml + mathml-to-latex 两个轻量包，
在 hybrid-docx-parser 中提取公式并注入：
1. styleAst 节点挂 `math_latex` 字段（精确按段落对齐）
2. Markdown 末尾追加公式附录（供 LLM 审阅）
