## 测试验证 (2026-03-15)

| 项目 | 结果 |
|------|------|
| 根路径 `/` 重定向 | 307 → `/zh`（Accept-Language 无 cookie 时） |
| 中文首页 `/zh` | 文案正确 |
| 语言切换 中文→EN | 跳转 `/en`，文案切换为英文 |
| Auth 页 `/en/login` | Sign in to ThesisAI 等英文文案正确 |
| Link locale 保持 | 链接自动带当前 locale |

## 路由结构

```
/ → /zh (默认)
/zh, /en → 首页
/zh/login, /en/login → 登录
/zh/register, /en/register → 注册
/zh/dashboard, /en/dashboard → 控制台
/auth/callback → OAuth 回调（无 locale 前缀）
```

## 注意事项

- Server Actions 的 `redirect()` 使用 `next/navigation`，依赖 middleware 将 `/dashboard` 重定向到 `/[locale]/dashboard`
- 页面 metadata 的 title 仍为中文，可按 locale 动态生成（后续优化）
