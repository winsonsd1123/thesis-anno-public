# 设置页修改密码（执行记录）

## 上下文

Dashboard 设置页增加「当前密码 + 新密码」直改；纯 OAuth（无 `email` identity）不展示表单。

## 实现要点

- `changePasswordInSettingsSchema`（`lib/schemas/auth.schema.ts`）
- `authService.changePasswordWithCurrentPassword`：`signInWithPassword` → `updateUserPassword`（DAL 原子方法复用）
- `changePasswordInSettings` Server Action（`lib/actions/auth.actions.ts`），成功不 redirect，`getTranslations` 报错文案
- `ChangePasswordForm.tsx` + `settings/page.tsx` 传入 `canChangePassword`
- `messages/zh.json`、`messages/en.json`：`dashboard.changePasswordForm`

## 验证

- `npx tsc --noEmit` 通过
- 建议手动：邮箱注册用户打开设置页改密；OAuth-only 用户不显示改密卡片
