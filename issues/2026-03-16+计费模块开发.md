## 环境变量

需在 `.env.local` 中配置：

- `SUPABASE_SERVICE_ROLE_KEY`（钱包/订单写入）
- Zpay：`ZPAY_PID`、`ZPAY_KEY`、`ZPAY_SUBMIT_URL`（https://zpayz.cn/submit.php）

## Zpay 对接与修复（2026-03-16 后续）

### 1. API 域名修正
- **问题**: `z-pay.cn` 调用 mapi 超时
- **修改**: 改用 `zpayz.cn`（ZPAY_MAPI_URL、ZPAY_SUBMIT_URL）

### 2. 支付方式改为 submit.php
- **问题**: mapi.php 网络超时
- **修改**: 按官方 demo 使用 submit.php 页面跳转，本地拼接 URL，无需 API 调用

### 3. return_url 修正
- **问题**: 文档要求 return_url 不支持带参数
- **修改**: 去掉 `?paid=1`，前端识别 `trade_status=TRADE_SUCCESS`

### 4. Webhook 调试
- 增加各失败分支日志
- 验签失败时输出 expected/received（非生产）

### 5. 备用支付链接
- PricingCard 在自动跳转失败时展示「点击打开支付页面」

### 6. 单次体验价格
- 测试用：price 改为 1 分（¥0.01）

### 7. 数据库 metadata JSON 修复
- **问题**: `add_credits_deposit` 中 metadata 字符串拼接导致 `invalid input syntax for type json`
- **修改**: 迁移使用 `jsonb_build_object('order_id', p_order_id::text)` 替代字符串拼接
